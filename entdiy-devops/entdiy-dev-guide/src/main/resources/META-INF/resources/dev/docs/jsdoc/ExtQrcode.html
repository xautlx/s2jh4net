Namespace: ExtQrcode

ExtQrcode

构造二维码显示: div[data-qrcode]
点击二维码自动转换为图片下载，方便线下使用

Properties:
Name Type Argument Default Description
data-qrcode string

二维码内容text文本，一般是一个绝对路径的URL
data-foreground color

二维码颜色
ata-qrcode-icon uri <optional>

中心小图标的URL路径，可选值，未提供则无图标显示
data-qrcode-header string <optional>

二维码上方显示标题文本
data-qrcode-footer string <optional>

二维码下方显示标题文本
data-convert-image boolean <optional>
 true

是否支持点击二维码转换为Image图片便于下载使用

Example

<div data-qrcode="${qrcodeUrlPrefix}/${entity.id}" data-qrcode-icon="${ctx}/assets/img/logo.png" data-qrcode-header="欢迎访问" data-qrcode-footer="扫码查看详情"/>
